## <a name="machine-learning"></a><span data-ttu-id="0372f-132">Machine Learning</span><span class="sxs-lookup"><span data-stu-id="0372f-132">Machine learning</span></span>
<span data-ttu-id="0372f-133">Machine Learning은 예측 모델을 학습하는 데 사용되는 수학 모델링 기술입니다.</span><span class="sxs-lookup"><span data-stu-id="0372f-133">Machine learning is a mathematical modeling technique used to train a predictive model.</span></span> <span data-ttu-id="0372f-134">일반적인 원리는 대규모 기록 데이터 집합에 통계 알고리즘을 적용하여 포함된 필드 간 관계를 파악하는 것입니다.</span><span class="sxs-lookup"><span data-stu-id="0372f-134">The general principle is to apply a statistical algorithm to a large dataset of historical data to uncover relationships between the fields it contains.</span></span>

<span data-ttu-id="0372f-135">일반적으로 Machine Learning 모델링은 데이터 과학자가 수행하며, 모델을 학습하기 전에 데이터를 철저히 탐색하고 준비해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="0372f-135">Machine learning modeling is usually performed by data scientists, who need to thoroughly explore and prepare the data before training a model.</span></span> <span data-ttu-id="0372f-136">이러한 탐색 및 준비에는 일반적으로 다양한 대화형 데이터 분석 및 시각화 작업이 수반됩니다. 즉, 이 작업을 위해 특수하게 디자인된 대화형 도구 및 환경에서 Python 및 R 등의 언어를 사용하게 됩니다.</span><span class="sxs-lookup"><span data-stu-id="0372f-136">This exploration and preparation typically involves a great deal of interactive data analysis and visualization &mdash; usually using languages such as Python and R in interactive tools and environments that are specifically designed for this task.</span></span>
