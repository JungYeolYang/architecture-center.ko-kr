<span data-ttu-id="4ea01-114">올바른 인증 방법을 선택하는 것은 앱에서 클라우드로 이동하려는 조직이 첫 번째로 고려해야 할 사항입니다.</span><span class="sxs-lookup"><span data-stu-id="4ea01-114">Choosing the correct authentication method is the first concern for organizations wanting to move their apps to the cloud.</span></span>

<span data-ttu-id="4ea01-115">이 방법을 선택하면 Azure AD에서 사용자의 로그인 프로세스를 처리합니다.</span><span class="sxs-lookup"><span data-stu-id="4ea01-115">When you choose this method, Azure AD handles users' sign-in process.</span></span> <span data-ttu-id="4ea01-116">원활한 SSO(Single Sign-On)와 결합되어 있으므로, 사용자는 자격 증명을 다시 입력하지 않고도 클라우드 앱에 로그인할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="4ea01-116">Coupled with seamless single sign-on (SSO), users can sign in to cloud apps without having to reenter their credentials.</span></span> <span data-ttu-id="4ea01-117">클라우드 인증을 사용할 경우 다음 두 가지 옵션 중에서 선택할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="4ea01-117">With cloud authentication, you can choose from two options:</span></span>
