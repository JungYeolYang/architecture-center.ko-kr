# Azure 참조 아키텍처

Azure 참조 아키텍처는 관련 아키텍처를 그룹화하여 시나리오별로 정리되어 있습니다.   각각의 아키텍처에는 확장성, 가용성, 관리 효율성, 보안에 대한 고려사항과 더불어 권장사항이 포함되어 있습니다. 또한 대부분의 아키텍처에는 배포 가능한 솔루션도 포함되어 있습니다.
